# Challenge-Alura-Store
## Challenge Alura Store - Grupo 9
### Descripción del Proyecto:
El Proyecto presenta un análisis de indicadores económicos y de funcionamiento de cuatro tiendas pertenecientes a la cadena Alura Store. El modelo de análisis aplicado permite seleccionar qué tienda es la opción recomendada para la venta, a partir de un análisis de facturación, análisis de calificación, análisis de productos vendidos por categorías, análisis de productos más y menos vendidos y análisis de costos de envíos de sus productos. Los resultados obtenidos del procesamiento de datos son visualizados en gráficos para facilitar la toma de decisiones y justificar la elección de la tienda.
Si bien este proyecto es aplicado a datasets de la cadena Alura Store, puede ser aplicado a otras fuentes de datos debido a la posibilidad de leer otros datasets presentados en las primeras líneas de código del Proyecto.
Finalmente se presenta un Informe Final, con introducción, desarrollo y conclusión, resumiendo la información obtenida del análisis.

### Tecnologías utilizadas en el Proyecto:
Python y librería Matplotlib
